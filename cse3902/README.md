To move Link - WASD

  Attack - Z or Enter key
  Take Damage - E (testing)

  Use Boomerang - 1
  Use Bomb - 2
  Use Arrow - 3
  
  Pause/Unpause - P
  Open/Close Inventory - I
  
  In Inventory:
        Item Selection - Arrow keys
        Equip item - Enter key

  Quit Game - Q(for testing purposes)
